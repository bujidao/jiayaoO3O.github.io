导数的含义

导数是增量比的极限.

这句话揭露了导数的本质是极限, 也说明了为什么第一章要学极限, 因为导数就是极限.

  • 用导数的定义求导
  • 复合函数求导
  • 高阶求导
  • 隐函数求导
  • 参数方程求导

用定义求导

导数的三大定义形式 :

f′(x0)=lim⁡Δx→0ΔyΔx=lim⁡Δx→0f(x0+Δx)−f(x)Δxf'\left(x_0\right)=\lim_{\Delta x\rightarrow0}\frac{\Delta y}{\Delta x}=\lim_{\Delta x\rightarrow0}\frac{f\left(x_0+\Delta x\right)-f\left(x\right)}{\Delta x} f′(x0​)=Δx→0lim​ΔxΔy​=Δx→0lim​Δxf(x0​+Δx)−f(x)​

f′(x0)=lim⁡h→0f(x0+h)−f(x0)hf'\left(x_0\right)=\lim_{ h\rightarrow0}\frac{f\left(x_0+h\right)-f\left(x_0\right)}h f′(x0​)=h→0lim​hf(x0​+h)−f(x0​)​

f′(x0)=lim⁡x→x0f(x)−f(x0)x−x0f'\left(x_0\right)=\lim_{ x\rightarrow x_0}\frac{f\left(x\right)-f\left(x_0\right)}{x-x_0} f′(x0​)=x→x0​lim​x−x0​f(x)−f(x0​)​

为什么要用定义求导, 而不是直接用求导公式, 那是因为使用求导公式之前题目必须已知f(x)f(x)f(x)可导, 可导可以推出连续, 连续可以推出极限, 但是反过来却不成立.

可导→连续→极限可导\rightarrow连续\rightarrow极限 可导→连续→极限

可导↚连续↚极限可导\nleftarrow连续\nleftarrow极限 可导↚连续↚极限

例如 :

设f(x)在x=2处连续,且lim⁡x→2f(x)x−2=2,求f′(2)设f(x)在x=2处连续, 且\lim_{x\rightarrow2}\frac{f\left(x\right)}{x-2}=2, 求f'(2) 设f(x)在x=2处连续,且x→2lim​x−2f(x)​=2,求f′(2)

题目只是说了在x=2x=2x=2处连续, 但是连续不一定可导, 所以不能用公式, 也不能用其他方法去解决, 只能用定义.

例如 :

设φ(x)在x=a处连续,f(x)=(x2−a2)φ(x),求f′(x)设\varphi\left(x\right)在x=a处连续, f(x)=(x^2-a^2)\varphi\left(x\right), 求f'(x) 设φ(x)在x=a处连续,f(x)=(x2−a2)φ(x),求f′(x)

题目只是说了连续, 但是连续不一定可导, 所以只能用定义来算.

f′(a)=lim⁡x→af(x)−f(a)x−af'\left(a\right)=\lim_{x\rightarrow a}\frac{f\left(x\right)-f\left(a\right)}{x-a} f′(a)=x→alim​x−af(x)−f(a)​

=lim⁡x→a(x2−a2)φ(x)−0x−a=\lim_{x\rightarrow a}\frac{\left(x^2-a^2\right)\varphi\left(x\right)-0}{x-a} =x→alim​x−a(x2−a2)φ(x)−0​

=lim⁡x→a(x+a)φ(x)=\lim_{x\rightarrow a}\left(x+a\right)\varphi\left(x\right) =x→alim​(x+a)φ(x)

=2aφ(a)=2a\varphi\left(a\right) =2aφ(a)

可以看出, 导数与极限的关系, 导数就是极限, 也遵守极限的计算规则.

用定义解决几何问题

用定义求导的另一个出题方向是解决几何问题, 主要是解决切线方程和法线方程.

法线就是与切线垂直的线, 法线斜率与切线斜率相乘为-1

例如 :

求曲线y=ex在点(0,1)处的切线方程和法线方程求曲线y=e^x在点(0, 1)处的切线方程和法线方程 求曲线y=ex在点(0,1)处的切线方程和法线方程

∵y′(0)=ex∣x=0=e0=1\because y'(0)=e^x\vert _{x=0}=e^0=1 ∵y′(0)=ex∣x=0​=e0=1

∴切线方程y−1=1⋅(x−0)\therefore 切线方程y-1=1\cdot(x-0) ∴切线方程y−1=1⋅(x−0)

即x−y+1=0即x-y+1=0 即x−y+1=0

法线方程y−1=−11⋅(x−0)法线方程y-1=-\frac 11\cdot (x-0) 法线方程y−1=−11​⋅(x−0)

即x+y−1=0即x+y-1=0 即x+y−1=0

复合函数求导

复合函数求导公式 :

y=f[g(x)]在点x可导,则其导数为dydx=f′(u)⋅g′(x)y=f[g(x)]在点x可导, 则其导数为 \frac{dy}{dx} = f'(u)\cdot g'(x) y=f[g(x)]在点x可导,则其导数为dxdy​=f′(u)⋅g′(x)

解题步骤 :

  1. 满足运算法则的的要优先运算.
  2. 数清楚有多少个函数复合, 这是最关键一步, 数错了就全错了.
  3. 有多少个函数复合就求多少次导, 由外向内, 从左向右.
  4. 每求一次导, 就扔掉一个函数.

例如 :

求函数y=(ln(tan(x2)))3的导数求函数y=\left(ln\left(tan\left(\frac x2\right)\right)\right)^3的导数 求函数y=(ln(tan(2x​)))3的导数

第一步先数清楚有多少个函数复合, 需要严格按照由外向内, 从左到右原则, 可以看到, 从按顺序分别为3次方3次方3次方, lnlnln, tantantan, x2\frac x22x​ 四个函数的复合, 所以计算的公式即为 :

y′=3⋅3(ln(tan(x2)))2⋅1tan⁡(x2)⋅sec⁡2(x2)⋅12y'=3\cdot 3\left(ln\left(tan\left(\frac x2\right)\right)\right)^2\cdot \frac1{\tan\left(\frac x2\right)}\cdot \sec^2\left(\frac x2\right)\cdot\frac 12 y′=3⋅3(ln(tan(2x​)))2⋅tan(2x​)1​⋅sec2(2x​)⋅21​

有四个函数, 所以要求4次导数, 严格按照由内向外, 从左到右的原则.

例如 :

求函数y=cos⁡(4−3x)的导数求函数y=\cos(4-3x)的导数 求函数y=cos(4−3x)的导数

第一步先数清楚有多少个函数复合, 从左到右, 分别为cos⁡\coscos, 4−3x4-3x4−3x 这两个函数, 所以按照步骤:

y′=−sin⁡(4−3x)⋅(4−3x)′y'=-\sin(4-3x)\cdot (4-3x)' y′=−sin(4−3x)⋅(4−3x)′

=−sin⁡(4−3x)⋅(−3)=3sin⁡(4−3x)=-\sin(4-3x)\cdot (-3)=3\sin(4-3x) =−sin(4−3x)⋅(−3)=3sin(4−3x)
